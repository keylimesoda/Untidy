# Untidy 0.5 beta — 2026-06-20

Untidy 0.5 beta is the first Ric-validated physical-watch build with the new Now Playing interaction model.

- GitHub release: to be created after this document is committed.

## What is validated
- Crown volume works on Now Playing.
- Now Playing hides grey unpopulated controls while track metadata loads.
- Pull-over context menu uses a grey guarded bottom grip instead of a button or fake page.
- Long-press arms the grip; the sheet follows upward drag 1:1.
- Menu surface is charcoal and uses the same handle visual/state for close.
- Menu handle is lowered to avoid the watch universal settings pulldown zone.
- Search handoff uses Wear RemoteInput and minimizes the prior voice/text/keyboard flash.
